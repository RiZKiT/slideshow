Getting Started

  1. Download Prototype 1.6, Scriptaculous 1.8.1 effects and Prototype UI core and extra tools for now. You can also choose to use protoculous from protopack (protoculous-effects is enough for the slideshow) as prototype and scriptaculous replacement
  2. Change this document to load your required javascript file(s).
  3. Add two or more of your own images to the /images folder.
  4. Customize the data object in the source of the <head> of this document with the filenames and captions of your images.
  5. Customize the <img> tag in the source of this document with the src and alt text of the default image for the show.
  6. Reload this page.
